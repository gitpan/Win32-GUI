=item new Win32::GUI::Pen(COLOR | %OPTIONS)

Creates a new Pen object.
Allowed %OPTIONS are:

  -style =>
    0 PS_SOLID
    1 PS_DASH
    2 PS_DOT
    3 PS_DASHDOT
    4 PS_DASHDOTDOT
    5 PS_NULL
    6 PS_INSIDEFRAME
  -width => number
  -color => COLOR
