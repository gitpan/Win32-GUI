=head3 Constructor

=over 4

=for html <A NAME="new_Win32_GUI_ProgressBar_PARE">

=item new Win32::GUI::ProgressBar(PARENT, %OPTIONS)

Creates a new ProgressBar object;
can also be called as PARENT->AddProgressBar(%OPTIONS).
Class specific %OPTIONS are:

    -smooth   => 0/1 (default 0)
        uses a smooth bar instead of the default segmented bar.
    -vertical => 0/1 (default 0)
        display progress status vertically (from bottom to top).

=for html <P>

=back

=head3 Methods

=over 4

=for html <A NAME="SetPos_VALUE_">

=item SetPos(VALUE)

Sets the position of the ProgressBar to the specified VALUE.

=for html <P>

=for html <A NAME="SetRange_MIN_MAX_">

=item SetRange([MIN], MAX)

Sets the range of values (from MIN to MAX) for the ProgressBar; if MIN
is not specified, it defaults to 0.

=for html <P>

=for html <A NAME="SetStep_VALUE_">

=item SetStep([VALUE])

Sets the increment value for the ProgressBar; see  L<StepIt()|/StepIt_>.

=for html <P>

=for html <A NAME="StepIt_">

=item StepIt()

Increments the position of the ProgressBar of the defined step value;
see  L<SetStep()|/SetStep_VALUE_>.

=for html <P>
