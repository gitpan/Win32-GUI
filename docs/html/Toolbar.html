Constructor

new Win32::GUI::Toolbar(PARENT, %OPTIONS)

Creates a new Toolbar object; can also be called as PARENT->AddToolbar(%OPTIONS). Class specific %OPTIONS are: 

  -flat      => 0/1
  -imagelist => IMAGELIST
  -multiline => 0/1
  -nodivider => 0/1
